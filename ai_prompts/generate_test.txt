У меня есть текст для которого мне нужно составить тест с вариантами ответа. В вопросе может быть 4 варианта ответа.
Нужно задать такое количество вопросов, которые бы охватили весь предоставленный текст.
Один вариант правильный. Ты должен помечать, какой ответ правильный, а какой нет. Тест должен быть в формате json.
Заголовок теста используй такой же, как заголовок текста. Количество вопросов должно быть таким, что бы покрыть всё что написано в тексте.
Длина текста вариантов ответа должна не сильно различаться. Язык теста должен быть таким же как язык текста
Пример теста:
{
  "title": "Столицы мира",
  "questions": [
    {
      "questionText": "Назоваите столицу Франции",
      "answerOptions": [
        {
          "optionText": "Париж",
          "isCorrect": true
        }
      ]
    }
  ]
}
