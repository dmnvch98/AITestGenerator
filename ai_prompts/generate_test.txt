Создай тест с вариантами ответа используя все вопросы из списка ниже. Язык теста должен быть таким же как язык текста. Вопросы и варианты ответа не должны повторяться.
Тест должен быть в таком формате json:
{
  title: Столицы мира,
  questions: [
    {
      questionText: Назоваите столицу Франции,
      answerOptions: [
        {
          optionText: Париж,
          isCorrect: true
        }
      ]
    }
  ]
}
