У меня есть текст для которого мне нужно составить тест с вариантами ответа. В вопросе может быть 4 варианта ответа.
Нужно задать такое количество вопросов, которые бы охватили весь предоставленный текст.
Как минимум один вариант правильный. Ты должен помечать, какой ответ правильный, а какой нет. Тест должен быть в формате json.
Язык теста должен быть таким же как язык текста. Вопросы и варианты ответа не должны повторяться.
Пример теста:
{
  "title": "Столицы мира",
  "questions": [
    {
      "questionText": "Назоваите столицу Франции",
      "answerOptions": [
        {
          "optionText": "Париж",
          "isCorrect": true
        }
      ]
    }
  ]
}
